# yadd/lemonldap-ng-webpubsub

Web Pub/Sub server fro [LemonLDAP::NG](https://lemonldap-ng.org)

## Variables

- `PUBSUG_ACCESS_LOG`, default STDOUT
- `PUBSUB_ADDR`
- `PUBSUB_CERT`
- `PUBSUB_DEBUG`
- `PUBSUB_KEY`
- `PUBSUB_MAX_REQUEST_SIZE`, default 8192
- `PUBSUB_MAX_REQUEST_TIME`, default 5
- `PUBSUB_QUIET`
- `PUBSUB_PORT`
- `PUBSUB_TOKEN`

LemonLDAP::NG — Helm charts

Helm repository for the guimard/llng-docker project: a Kubernetes chart deploying the LemonLDAP::NG portal, manager and maintenance cronjob.

Quick start

helm repo add llng https://guimard.github.io/llng-docker/helm/latest
helm repo update
helm install my-release llng/lemonldap

For the 2.21 LTS line, use the lts-2.21 channel instead:

helm repo add llng-lts https://guimard.github.io/llng-docker/helm/lts-2.21
helm repo update
helm install my-release llng-lts/lemonldap

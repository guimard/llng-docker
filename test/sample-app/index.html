Bienvenue sur l'application protegee

Si vous voyez cette page, c'est que vous etes correctement authentifie via LemonLDAP::NG.

Cette application est protegee par le SSO LemonLDAP::NG. Toute requete non authentifiee est automatiquement redirigee vers le portail d'authentification.

Vous pouvez maintenant integrer votre propre application web dans ce conteneur en remplacant les fichiers dans sample-app/.
